Webhook API Documentation

Overview

The webhook system for Onemetric enables you to receive real-time notifications about key events, such as market alerts and news updates. This document provides all necessary details for integrating and using the webhook API.

Endpoint Details

API URL https://your-webhook-url
Method POST
Content Type application/json
Authorization API key or Token in the headers

Request Contract

The payload for the webhook request must conform to the following JSON structure:

{
  "event_type": "MARKET_GENERATED_ALERT",
  "sub_type": "MARKET_PREV_HIGH_ALERT",
  "time_stamp": "2024-10-01T10:20:00Z",
  "details": {
      "ltp": 30,
      "close": 20,
      "high": 35,
      "low": 15
  }
}

The following event_type values are supported:

  • MARKET_GENERATED_ALERT
  • NEWS_ALERT

MARKET_GENERATED_ALERT

This event type includes all technical alerts generated based on real-time market data. Supported sub_type values include:

  • MARKET_PREV_HIGH_ALERT: Generated when the current stock price exceeds the previous day's high.
  • RSI_OVERSOLD: Generated when the stock's RSI (Relative Strength Index) value indicates an oversold condition.
  • RSI_OVERBOUGHT: Generated when the stock's RSI (Relative Strength Index) value indicates an overbought condition.
  • MARKET_AVG_20_ALERT: Generated based on the 20-day Simple Moving Average (SMA) of the stock's closing price.
  • MARKET_AVG_50_ALERT: Generated based on the 50-day Simple Moving Average (SMA) of the stock's closing price.

NEWS_ALERT

This event type is generated when significant news or updates related to a stock are identified. These alerts help keep you informed of critical developments.

Response

A successful webhook request will return the following response:

{
  "status": "success",
  "message": "Webhook received successfully"
}

Notes

Important: Ensure your webhook URL is publicly accessible and uses HTTPS for secure communication.
